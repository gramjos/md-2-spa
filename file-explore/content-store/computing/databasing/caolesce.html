COALESCE function in SQL Server returns the first non-NULL expression among its arguments. COALESCE(MiddleName, 'N/A') checks if MiddleName is NULL. If it is, it returns 'N/A'; otherwise, it returns the MiddleName value.

sql
SELECT 
    FirstName,
    LastName,
    COALESCE(MiddleName, 'N/A') AS MiddleName
FROM 
    Employees;

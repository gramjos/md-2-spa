Callouts

Callouts are foldable boxes (when exported to .png they remain open). The box color and icon can be customized. Sources: Zolt's HowTo videoand Obsidian Docu

Making a Callout

[!NOTE] Title Contents

After the icon declaration closing bracket, a '-' or '+' dictates the default behavior for open or close callout.

To enable custom CSS

In Settings > Appearance > CSS Snippets custom_callout_icons.css modifies CSS_Variables $ cat custom_callout_icons.css | head -c 250

css
.callout[data-callout="cci"] {
    --callout-color: 0, 0, 0;
    --callout-icon: '<svg version="1.1" id="Layer_1" xmlns="http://www.w3.org/2000/svg" xmlns:xlink="http://www.w3.org/1999/xlink" x="0px" y="0px" width="100%" viewBox="0 0 84 94" enable-ba // cont. // ...

Excalidraw Created Icons

Create simple graphic. Once drawing is complete, then open the drawing in its markdown mode to add the following tags.

markdown-tags
excalidraw-export-svgpadding: 0
excalidraw-export-transparent: true
excalidraw-export-dark: false

In Excalidraw then export them as SVG. Upload the Excalidraw SVG to https://jakearchibald.github.io/svgomg/ to minify by turning off all switches that do not remove the image from the screen. About 20% reduction in file should be expected.

In my .obsidian.vimrc I have the two lines

vim
exmap addLineDown jsfile vim_tricks/addLineSpace.js {addLine('down')}
map <Space>j :addLineDown

The first line imports the resource (the external javascript file) and assigns it to the variable addLineDown and the second line maps the resource to vim bindings. In the Obsidian Vault directory there lies a the file addLineSpace.js inside the folder vim_tricks and the entirety of addLineSpace.js is below

js
function addLine(spaceDirection) {
	const lineNo = editor.getCursor().line; 
	const charNo = editor.getCursor().ch; 
  if (spaceDirection=="down"){
    editor.setCursor(lineNo)
    editor.newlineOnly(); 
  }else{
    editor.setCursor(lineNo-1)
    editor.newlineOnly(); 
  }
  editor.setCursor(lineNo, charNo); 
  return; 
 }

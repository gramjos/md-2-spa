After scouring the api function list we find setActiveTool f(t) Then search the official repo the function name 'setActiveTool' official repo: "https://github.com/excalidraw/excalidraw" search query URL "https://github.com/search?q=repo%3Aexcalidraw%2Fexcalidraw+setActiveTool&type=code" Also, note, that api.getAppState() yields back an object containing an object called activeTool and it contains

js
activeTool:
customType: null
lastActiveTool: null
locked: false
type: "selection"

Now turn the .excalidraw drawing into .md view mode and add the property excalidraw-onload-script

md
---
excalidraw-onload-script: "ea.getExcalidrawAPI().setActiveTool({type:'hand'})" 
---

the above javascript embedded in the string assumes the lines

js
ea=ExcalidrawAutomate
ea.setView()
